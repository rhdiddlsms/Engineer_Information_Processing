# XML
HTML의 단점을 보완한 인터넷 언어로, SGML의 복잡한 단점을 개선한 특수 목적 마크업 언어

# WSDL 
메시지 포맷, 프로토콜 정보 등 웹 서비스에 대한 상세 정보가 기술된 XML 형식으로 구현되어 있는 언어

# 데이터 마이닝 
대규모로 저장된 데이터 안에서 통계적 규칙이나 패턴 등을 발견하고 이를 의미 있는 정보로 변환하는 기술

# 인스펙션 
원시 코드 등의 저작자 외의 다른 전문가 또는 팀이 검사하는 검토 기법

# 워크스루 
검토 자료를 회의 전에 배포해서 사전에 검토를 진행한 후 짧은 시간 동안 회의를 통해 코드의 오류를 검출하고 문서화하는 비공식적 기법

# ESB
이기종 및 앱들 간을 연계해서 관리 운용 할 수 있도록 서비스 중심의 통합을 지향하는 기술

# 정규화 
관계형 데이터베이스의 설계에서 중복이 최소화 될 수 있도록 데이터를 구조화하여 이상현상이 생기지 않도록 하는 데이터 모델링 기법

# 티어 드롭
IP 패킷의 재조합 과정에서 잘못된 Frament Offset 정보로 인해 단편화된 패킷의 재조합 과정에서 문제를 발생하도록 만드는 공격

# 로킹 기법
하나의 트랜잭션에서 데이터를 액세스 하는 동안, 다른 트랙잭션이 그 데이터 항목을 액세스 할 수 없도록 제어하는 기법

# 병행 제어
동시에 여러개의 트랜잭션을 수행할 때 데이터베이스의 일관성 유지를 위해서 트랜잭션 간의 상호 작용을 제어하는 기법

# 트랜잭션 
하나의 논리적 기능을 정상적으로 수행하기 위한 작업의 기본 단위

# 테일러링 
프로젝트의 특성과 필요에 따라 프로세스, 기법, 산출물을 요구에 맞도록 최적화 하는 방법론

# 온톨로지 
개체, 속성, 관계 정보를 컴퓨터가 이해할 수 있도록 서술한 개념화 명세서

# 시맨틱 웹 
관계-정보를 컴퓨터가 처리할 수 있는 온톨로지 형태로 표현하고, 자동화된 기계가 처리하도록 하는 지능형 웹

# 프로시저 
일련의 SQL 쿼리의 집합으로 마치 하나의 함수처럼 쿼리의 집합을 실행하여 데이터를 조작하는 프로그램

# 파티셔닝 
성능 향상을 위해 대용량의 데이터베이스를 작은 단위로 나누는 기술

# 테스트 오라클 
테스트 수행 결과가 참/거짓인지 판단하기 위해 미리 정의된 참 값을 대입하여 비교하는 기법

# 동치 분할 테스트
유사한 도메인 별로 유효값/무효값을 그룹핑하여 대표값을 테스트 케이스로 도출하는 기법

# NoSQL
고정된 테이블 스키마가 필요하지 않고, 수평적으로 확장이 가능한 DBMS

# VPN
인터넷과 같은 공중망에 터널링을 사용해서 사설망을 제공하는 가상 네트워크

# REST
HTTP URI를 통해 자원을 명시하고 HTTP 메소드를 통해 자원에 명령을 적용하는 소프트웨어 아키텍처

# Alien Code 
아주 오래 되거나 참고 할 문서 혹은 개발자가 없어 유지보수가 어려운 나쁜 코드

# 시스템 카탈로그
데이터 개체들에 대한 모든 정보와 명세가 저장되어 있는 테이블의 집합체

# 스머프 공격
출발지 주소를 공격 대상의 IP로 설정하여 네트워크 전체에 ICMP Echo 패킷을 직접 브로드캐스팅 하여 타겟 시스템을 마비시키는 공격 기법

# IPSec
IP계층에서 무결성과 인증을 보장하는 인증헤더(AH)와 기밀성을 보장하는 암호화(ESP)를 이용하여 양 종단 간 구간에 보안 서비스를 제공하는 터널링 프로토콜

# SSL
전송계층과 응용계층 사이에서 클라이언트와 서버 간의 웹 데이터 암호화 및 전송 시 데이터 무결성을 보장하는 보안 프로토콜

# 타임 스탬프 오더링
고유번호인 시간 스탬프를 트랜잭션에 부여해 트랜잭션 순서를 미리 선택하는 기법

# 낙관적 검증 
트랜잭션을 수행하는 동안 어떠한 검사도 하지 않고, 트랜잭션 종료 시 일괄적으로 검증을 수행

# 옵티마이저
SQL문을 처리 할 수 있는 최적의 실행 계획을 수립하는 DBMS 핵심 엔진

# 보안 운영체제
운영체제의 커널에 보안 기능 및 참조 모니터를 추가하여 해킹으로부터 보호하는 운영체제

# SDN(Software Defined Network) 
오픈 API를 기반으로 컨트롤 플레인과 데이터 플레인을 분리한 소프트웨어 기반의 네트워크 기술

# NFV(Network Function Virtualization)
네트워크 기능을 가상 기능으로 모듈화하고 필요한 곳에 기능을 제공하는 네트워크 가상화 기술

# BCP(Business Continuity Planning) 
재해복구, 업무복구 및 재개, 비상계획 등을 통해 비즈니스 연속성을 보장하는 체계

# RTO(Recovery Time Objective) 
업무가 복구되어 다시 가동 될 때 까지의 시간

# RPO(Recovery Point Objective) 
데이터의 손실 허용 시점

# BIA(Business Impact Analysis) 
영향도 및 손실 평가를 조사하는 BCP를 구축하기 위한 비즈니스 영향 분석

# DRS(Disaster Recovery System) 
재해복구 계획의 원할한 수행을 지원하는 재해복구 센터

# 결합도
외부 모듈과의 연관도를 나타냄, 모듈 간 상호 의존성을 나타냄

# 응집도 
모듈 내부 구성요소 간 연관 정도, 모듈의 독립성을 나타냄

# 데이터 흐름도 
시스템 분석과 설계에서 매우 유용하게 사용되는 다이어그램

# 자료 사전 
자료 요소, 자료 요소들의 집합, 자료의 흐름, 범위, 단위 등을 구체적으로 명시하는 사전

# HIPO 
시스템의 분석 및 설계나 문서화 할때 사용되는 하향식 소프트웨어 개발을 위한 문서화 도구

# 고가용성 
서버, 네트워크 등이 장애에 대응하여 긴 시간 동안 지속적으로 정상 운영이 가능한 상태

# tpmc 
하드웨어 성능 지표로 사용되며, 1분당 최대 처리 건수를 나타낸 수치

# 단일 책임의 원칙 
하나의 클래스는 하나의 역할만을 하도록 설계

# CRUD Matrix 
프로세스와 데이터 사이에 관계 의존성을 CRUD로 표현한 매트릭스

# 인덱스 
검색 연산의 최적화를 위해 데이터베이스 내의 칼럼에 대한 정보를 구성해 놓은 자료구조

# 메타 데이터 
자료들의 속성, 정보들을 담고있는 데이터의 데이터

# 테스트 케이스 
특정 요구사항에 준수하는지를 확인하기 위해 개발된 입력 값, 실행 조건, 예상된 결과의 집합

# FP 
요구기능 요인별 가중치를 합산, 총 기능 점수 계산, 비용산정 모형

# 소프트웨어 아키텍처 
소프트웨어 구성요소와 구성요소들 간의 관계를 나타낸 시스템 구조

# API 
운영체제나 프로그래밍 언어가 제공하는 기능을 제어 할 수 있게 만든 인터페이스

# IGMP 
호스트 컴퓨터와 인접 라우터가 멀티캐스트 그룹 멤버십을 구성하는 데 사용하는 통신 프로토콜

# 오류-부재의 궤변 
요구사항을 충족시켜주지 못한다면, 결함이 없다고 해도 품질이 높다고 볼 수 없는 소프트웨어 테스트 원리

# 슈퍼키 
모든 튜플에 대해 유일성은 만족시키지만 최소성은 만족시키지 못하는 키

# 관계 대수 
원하는 정보와 그 정보를 어떻게 유도하는가를 기술하는 절차적 언어

# 맵 리듀스 
대용량 데이터 처리를 분산 병렬 컴퓨팅에서 처리하기 위한 목적으로 제작한 소프트웨어 프레임워크

# 그룹 함수 
테이블의 전체 행을 하나 이상의 컬럼을 기준으로 컬럼 값에 따라 그룹화 하여 그룹별로 결과를 출력하는 함수

# 스레싱 
페이지 부재가 발생하여 프로세스의 처리 시간 보다 페이지 교체 시간이 더 많아지는 현상

# LOC 
각 기능의 원시 코드 라인 수의 낙관치 중간치 비관치를 측정하여 예측지를 구하고 이름 이용하여 비용을 산정하는 비용산정 모델

# 은행가 알고리즘 
사전에 필요한 자원의 수를 제시하고, 안정 상태 일 때만 자원을 프로세스에게 할당하는 교착상태 회피 기법

# 링크 상태 알고리즘 
다익스트라 알고리즘을 이용하여 링크 상태 정보를 모든 라우터에 전달하여 최단 경로 트리를 구성하는 라우팅 프로토콜 알고리즘

# 와이어 프레임 
UI 화면 구성을 협의하거나 서비스의 간략한 흐름을 공유하기 위해 화면 단위의 레이아웃을 설계하는 작업

# UML
객체지향 소프트웨어 개발 과정에서 생성된 산출물을 명세화, 시각화, 문서화 할 때 사용되는 모델링 언어

# 드라이브 바이 다운로드 
불특정 웹 서버와 웹 페이지에 악성 스크립트를 설치하고 사용자 접속 시 동의 없이 실행되어 의도된 멀웨어 서버로 연결되어 사용자를 감염 시키는 공격

# 멀티캐스트 프로토콜
